# Web Programming with JavaScript, HTML & CSS — Course Materials

Welcome! These are the materials for the 5-day introduction to web programming
(August 2026, live online).

## Start here

1. **Before Day 1:** work through the [Setup Guide](learner-guide/setup-guide.md)
   (~20 minutes — editor, one extension, a couple of settings).
2. Read the [Learner Guide](learner-guide/README.md) — what we cover each day and
   how to use these folders.
3. Something broken during the week? → [Troubleshooting](learner-guide/troubleshooting.md).

## Getting the files

Click the green **Code** button above → **Download ZIP**, then unzip into your
`webcourse` folder. The materials are updated at every break during the course —
re-download whenever the tutor says so (your own work in `my-work/` is safe as long
as you unzip *alongside* it, not over it).

## What's here

| Folder | Contents |
|---|---|
| `day1/` … `day5/` | Each day: `README`, `CHEATSHEET`, `demos/`, `exercises/`, `solutions/`, and (from Day 2) a `starter/` to catch up from |
| `learner-guide/` | Setup, troubleshooting, and the take-home guides |

Solutions etiquette: try first, peek second — and if you peek, *type* it, don't paste it.
