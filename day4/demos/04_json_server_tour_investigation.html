A tour of json-server

Start the server first, in the folder that contains db.json:

npx json-server@0.17.4 --watch db.json --port 3000

Leave it running. Now click the links below - each one is a GET request your browser makes just by visiting a URL. No code needed yet.

The menu

  • /members - the whole list
  • /members/1 - just Aidan
  • /members/99 - nothing: a 404 ("no such thing")

Query parameters (instructions after a ?)

  • /members?_sort=name - sorted by name
  • /members?active=true - only active members
  • /members?q=al - search for "al"
  • /members?_page=1&_limit=3 - the first three

DevTools moment: the Network tab

Press F12, open the Network tab, then reload /members. Click the members row and read the method (GET), the status (200), and the response (the JSON). Next block we write fetch, which does exactly this from code.

The file is the database: open db.json in VS Code, change a name by hand, save, then reload /members in the browser - your edit is there. Nothing magic underneath.
