REST verbs mapped to CRUD

Every app Creates, Reads, Updates and Deletes its data. On the web each of those is an HTTP verb plus a URL - "do this to that".

CRUD Verb Example request Meaning
Create POST POST /members add a new member
Read GET GET /members fetch the whole list
Read GET GET /members/3 fetch just member #3
Update PUT PUT /members/3 replace member #3
Delete DELETE DELETE /members/3 remove member #3

The verb is the action; the URL is the thing. GET only ever reads - which is why your browser can do it just by visiting a URL, as we'll see next in demo 04.
