What is an API?

An API is a program your program talks to by sending messages, instead of by clicking buttons. Today our Member Manager stops keeping its members in a hand-typed list and starts asking a server for them.

The waiter, not the kitchen

Our page is the customer. A server somewhere holds the data - that's the kitchen. We never go into the kitchen; we talk to the waiter (the API). The menu is the list of URLs we're allowed to ask for. We order in a shared language (HTTP), and a result comes back - usually as JSON.

Our kitchen this week

A tiny practice server called json-server, running on your own machine. It turns a plain file, db.json, into a real web API. You'll meet its menu in demo 04:

  • http://localhost:3000/members - the whole list
  • http://localhost:3000/members/1 - just Aidan
