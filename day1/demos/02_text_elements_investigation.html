Paragraphs and whitespace

This is a paragraph. Notice something important: no matter how much space or how many line breaks we type in the file, the browser COLLAPSES it all to single spaces. Layout is controlled by elements and (from this afternoon) CSS - never by mashing the space bar.

Need a line break inside a paragraph? There is
the br element - but if you are using lots of them, you probably wanted separate paragraphs or a list instead.

Emphasis - meaning, not looks

Use strong for text with strong importance, and emphasis for stressed wording. Yes, they LOOK like bold and italic - but they carry MEANING: a screen reader changes its tone of voice for them. (This "say what you mean" idea pays off big on Day 5.)
