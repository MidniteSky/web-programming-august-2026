Three ways to apply CSS

Way 1: I am styled by an inline style attribute. (Avoid this.)

Way 2: embedded stylesheet

The green h2 above is styled by the <style> block in this page's head. Better - but locked inside one page.

Way 3: external stylesheet

Anything with the class "external-demo" is styled by the separate file 07_styles.css - which could just as easily style a thousand pages.

Our rule from now on: all CSS lives in external .css files. One place to look, one place to change, shared by the whole site.
