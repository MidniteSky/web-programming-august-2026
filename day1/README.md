# Day 1 — How the Web Works: HTML & First CSS

Welcome! This folder contains everything for Day 1.

## Before we start (setup checklist)

1. **A modern browser** — Chrome, Edge or Firefox.
2. **Visual Studio Code** — free download from <https://code.visualstudio.com>.
3. **Live Server extension** — in VS Code: Extensions panel (Ctrl+Shift+X), search
   "Live Server" (by Ritwick Dey), click Install.
4. Open this `day1` folder in VS Code: *File → Open Folder…*

To view any `.html` file: right-click it in VS Code and choose **"Open with Live Server"**.
The page opens in your browser and refreshes automatically whenever you save.

## What you'll be able to do by the end of today

- Explain how a browser gets and displays a web page
- Write a correct HTML page from scratch: headings, text, lists, links, images, tables and forms
- Structure a page with semantic HTML (`header`, `nav`, `main`, `footer`)
- Style pages with an external CSS file: colours, fonts and text styling
- Inspect any website with the browser DevTools Elements panel

**End-of-day milestone:** a three-page *Member Manager* website — the project we will
keep improving every day this week.
