Units: px, rem, %, vh

rem - type that scales together

0.875rem (14px)

1rem (16px) - the base size

1.5rem (24px)

2rem (32px)

% - a fraction of the parent

width: 50% of my parent

vh - a fraction of the window height

height: 33vh - resize the window and watch me

Try it: in DevTools, find the html element and change font-size from 16px to 20px. Every rem-sized thing on the page grows in proportion - one dial for the whole page. That is why accessibility folk love rem: it respects a user who has set a larger default font.
