A card

Every blue on this page - the heading, this border, the button - comes from a single variable, --primary.

A button

Try it: in DevTools, select the html element (that's :root) and change --primary to seagreen in the Styles pane. Watch the heading, the border and the button all change together, instantly. That is the whole point - and exactly how we'll set up the Member Manager stylesheet so re-skinning it later is a one-line job.
